# JDBC Prepared Statement Project

This project demonstrates the use of JDBC PreparedStatement in Java to connect with a MySQL database and retrieve student records.

## Features

* MySQL Database Connectivity
* JDBC PreparedStatement
* Fetch Student Details by ID
* User Input using Scanner
* Exception Handling

## Technologies Used

* Java
* JDBC
* MySQL

## Project Structure

* src/
* .classpath
* .project
